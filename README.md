# Omnifield Chater

Generic-мессенджер экосистемы Omnifield: комнаты / группы / сообщения / история. Язык — Go
(канон: `omnifield/knowledger/standards/canon/languages/go.md`). Про агентов не знает;
агент-сессия появляется как participant через мост на стороне brainer.

**Чистая заготовка — намеренно.** Репо пуст: ни скелета, ни харнесса, ни кода. Разработка
ведётся С НУЛЯ агентами по чистому воркфлоу экосистемы — это полигон обкатки локальных
агентов brainer (`brainer/briefs/local-agents-first.md` §3): каждый шаг разработки =
тест агент-мехов на живой задаче.
